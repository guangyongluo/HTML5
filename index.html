文本标签

常用的文本标签：

b标签：标记包含内容，从语义上来看，就是标记一段文字。例如：这是HTML5教程；
strong标签：加粗包含内容，从语义上来看，就是强调一段重要的文本。例如：这是HTML5教程；
br标签：从语义上来看，就是强制换行；
wbr标签：从语义上来看，就是安全换行，主要针对英文单词在屏幕大小变化时换行的位置标识；
i标签：倾斜包含内容，表示外文词汇或科技术语，从语义上来看，表示区分周围内容，并不是特别强调或重要性。例如：这是HTML5教程；
em标签：倾斜包含内容，表示对外文词汇或科技术语加以强调。例如：这是HTML5教程；
s标签：删除包含内容，从语义上来看，表示不准确的删除（不强调）。例如：这是HTML5教程；
del标签：删除包含内容，从语义上来看，表示删除相关文字（强调）。例如：这是HTML5教程；
u标签：包含内容添加下划线，从语义上来看，并不强调此段文本。例如：这是HTML5教程；
ins标签：包含内容添加下划线，从语义上来看，强调此段文本。例如：这是HTML5教程；
small标签：将包含内容字体放小一号。例如：这是HTML5教程；
sup标签：数学的上标。例如：x的平方是x2；
sub标签：数学的下表。例如：x的以2为底的幂log2x；
code标签：表示计算机代码片段；
var标签：表示编程语言中的变量；
samp标签：表示程序或计算机的输出；
kdb标签：表示用户的输入；以上4个属性这属于英文范畴的，必须将 lang="en"英语才能体现效果。
abbr标签：从语义上看，是一段文本的缩写；
dfn标签：从语义上看，表示解释一个词或短语的一段文本；
q标签：从语义上来看，表示引用来自其他地方的内容；
cite标签：从语义上来看，表示引用其他作品的标题；
rudy标签：用来为非西方语言提供支持； 例如： 饕(tāo) 餮(tiè)
bdo标签：必须使用属性 dir 才可以设置，一共两个值：rtl（从右到左）和 ltr（从左到右）。一般默认是 ltr；
mark标签：实际作用就是加上一个黄色的背景，黑色的字；从语义上来看，与上下文相关而突出的文本，用于记号；
time标签：没有实际作用；从语义上来看，表示日期和时间；
span标签：没有实际作用；语义上就是表示一段文本。

最重要的文本标签<a>:

<a>元素属于文本元素，有一些私有属性或者叫局部属性。那么，相对应的还有通用属性或叫做全局属性。在这几个属性当中，只有 href 和 target 一般比较常用，而 href 是必须要用的。 解释：href 是必须属性，否则元素就变成空元素了。如果属性值是 http://开头的 URL，意味着点击跳转到指定的外部网站。target 属性告诉浏览器希望将所链接的资源显示在哪里。最常用的是_blank，新建一个窗口，而_self 是默认当前窗口显示。 相对与绝对路径：所谓相对路径，就是相对于链接页面而言的另一个页面的路径。而绝对路径，就是直接从 file:///磁盘符开始的完整路径。相对路径的条件是必须文件都在一个磁盘或目录下，如果是在同一个目录下，直接属性值就是被链接的文件名.后缀名。如果在同一个主目录下，有多个子目录层次，那就需要使用目录结构语法。

分组元素

p标签：建立段落，元素实际作用就是将内部包含的文本形成一个段落；而段落和段落之间保持一定量的空隙；
div标签：建立通用分组，在早期的版本中非常常用，通过<div>这种一般性分组元素进行布局；
blockquote标签：引用大段他处内容，语义上表示，大段的引用他处的内容；
pre标签：实际作用就是编辑器怎么排版的，原封不动的展现出来。当然，这种只适合简单的排版，复杂的排版就无法满足要求了；
hr标签：就是添加一条分割线，意图呈现上下文主题的分割；
ul li标签：<ul>元素表示无序列表，而<li>元素则是内部的列表项；
ol li标签：<ol>元素表示有序列表，而<li>元素则是内部的列表项。<ol>元素目前支持三种属性。start:从第几个序列开始统计；reversed:是否倒序排列；type:表示列表的编号类型，值分别为：1、a、A、i、I；
dl dt dd标签：生成说明列表；
figure figcaption标签：这两个元素一般用于图片的布局。

表格元素

表格的基本构成最少需要三个元素：<table>、<tr>、<td>，其他的一些作为可选辅助存在。
table tr td标签：构建基础表格，<table>元素表示一个表格的声明，<tr>元素表示表格的一行，<td>元素表示一个单元格。默认情况下表格是没有边框的，所以，在<table>元素增加一个border属性，设置为1即可显示边框；
th标签：为表格添加标题单元格，实际作用就是将内部文字居中且加粗；
thead标签：添加表头，尤其是以后动态生成表头，它的位置是不固定的，使用此元素可以限定在开头位置；
tfoot标签：添加表脚，限制在表格的底部；
tbody标签：添加表主体，主要是包含住非表头表脚的主体部分，有助于表格格式的清晰，更加有助于后续 CSS 和 JavaScript 的控制；
caption标签：添加表格标题；
colgroup标签：处理某个列，span 属性定义处理哪些列。1 表示第一列， 2 表示前两列。如果要单独设置第二列，那么需要声明两个，先处理第一个，将列点移入第二位，再处理第二个即可；
col标签：更灵活的设置列，表示单独一列，一个表示一列，控制更加灵活。如果设置了 span 则控制多列；

文档元素

文档元素的大部分标签，是没有任何效果的，完全是为了配合语义使用，以强调它的结构性。需要样式表CSS配合使用才能起到布局和样式的效果。
h1~h6标签：表示标题；<h1>~<h6>实际作用就是加粗并改变字体大小。用于各种标题文档。
header标签：元素主要设置页面的标头部分；
footer标签：元素主要设置页面的尾部；
hgroup标签：元素的作用就是当多个标题出现，干扰到一对或多个本身需要整合的标题，这是使用此元素包含群组；
section标签：元素的作用是定义一个文档的主题内容；
nav标签：元素给文档页面添加一个导航；
article标签：元素表示独立成篇的文档，里面可以包含头、尾、主题等一系列内容。在比较大的页面中会使用到，比如一片博文的列表，每篇博文，都有自己的头、尾、主题等内容。和此相似的还有论坛的帖子、用户的评论、新闻等；
aside标签：元素专门为某一段内容进行注释使用；
address标签：表示文档或article元素的联系信息；
details标签：元素生成详情区域;summary标签：元素在其内部生成说明标签。

嵌入元素

img标签：元素主要是插入一张外部的图片；其下的 src 属性表示初始化时显示的页面，width和height表示内嵌文档的长度和高度，ismap创建服务器端分区响应图，usemap 关联元素；
map标签：创建分区响应图；
iframe标签：表示内嵌一个HTML文档。其下的src属性表示初始化时显示的页面，width和height表示内嵌文档的长度和高度，name表示用于target的名称；
embed标签：元素是扩展浏览器的功能，大部分用于添加对插件的支持。可以添加了一个flash视频。type类型表示被插入内容的类型，width 和 height 表示宽高；src 表示文件路径；
progress标签：元素可以显示一个进度条，value表示进度条的进度值，max表示进度条的最大值，一般通过JS控制内部的值；
meter标签：元素显示某个范围内的值。其下的属性包含：min 和 max 表示范围边界，low表示小于它的值过低，high表示大于它的值过高，optimum表示最佳值，但不出现效果。

视频和音频元素

video标签：插入一个视频，主流的视频为.webm，.mp4，.ogg等。src表示资源URL；width表示宽度；height表示高度。autoplay表示自动开始播放；controls表示显示播放控件；loop表示循环播放；muted表示静音。preload 属性有三个值：none表示播放器什么都不加载；metadata表示播放之前只能加载元数据（宽高、第一帧画面等信息）；auto表示请求浏览器尽快下载整个视频，poster属性表示在视频的第一帧，做一张预览图；
audio标签：和video元素一样，audio元素用于嵌入音频内容，而音频元素的属性和视频元素类似。

表单元素

form标签：元素主要是定义本身是一组表单，元素属性action表示表单提交的页面；method表示表单的请求方式：有POST和GET两种，默认GET；enctype表示浏览器对发送给服务器的数据所采用的编码格式。有三种：application/x-www-form-urlencoded（默认编码，不能将文件上传到服务器）、multipart/form-data（用于上传文件到服务器）、text/plain（未规范的编码，不建议使用，不同浏览器理解不同），name设置表单名称，以便程序调用，target设置提交时的目标位置：_blank、_parent、_self、_top，autocomplete设置浏览器记住用户输入的数据，实现自动完成表单。默认为on自动完成，如果不想自动完成则设置off，novalidate设置是否执行客户端数据有效性检查；
input标签：元素默认情况会出现一个单行文本框，有五个属性，分别是autofocus让光标聚焦在某个input元素上，让用户直接输入，disabled禁用input元素，autocomplete单独设置input元素的自动完成功能，form让表单外的元素和指定的表单挂钩提交，name定义 input 元素的名称，以便接收到相应的值；
label标签：元素可以关联input，让用户体验更好。且更加容易设置CSS样式；
fieldset标签：元素可以将一些表单元素组织在一起，形成一个整体。元素属性name给分组定义一个名称；form：让表单外的分组与表单挂钩；disabled：禁用分组内的表单；
legend标签：元素给分组加上一个标题；
button标签：元素添加一个按钮，type 属性有如下几个值：submit表示按钮的作用是提交表单，默认值；reset表示按钮的作用是重置表单；button表示按钮为一般性按钮，没有任何作用。对于 type 属性为 submit 时，按钮还会提供额外的属性：form指定按钮关联的表单；formaction覆盖form元素的action属性；formenctype覆盖form元素的enctype属性；formmethod覆盖form元素的method属性；formtarget覆盖form元素的target属性；formnovalidate覆盖form元素的novalidate属性；
input标签的type属性
1. 当type值为text时，呈现的就是一个可以输入任意字符的文本框，这也是默认行为。并且，还提供了一些额外的属性。list指定为文本框提供建议值的datalist元素，其值为datalist元素的id值，maxlength设置文本框最大字符长度，pattern用于输入验证的正则表达式，placeholder输入字符的提示，readonly文本框处于只读状态，disabled文本框处于禁用状态，size设置文本框宽度，value设置文本框初始值，required表明用户必须输入一个值，否则无法通过输入验证；
2. 当type值为password时，一般用于密码框的输入，所有的字符都会显示星号。密码框也有一些额外属性和上面type为text一样；
3. 当type为search时，和文本框一致，在除Firefox浏览器的其他现代浏览器，会显示一个叉来取消搜索内容。额外属性也和text一致；
4. 当type为number、range时只限输入数字的文本框，不同浏览器可能显示方式不同。生成一个数值范围文本框，只是样式是拖动式，与上面几个type属性相比主要多了step属性，表示数字使用上下箭头增加的步长；
5. 当type为date系列时，实现文本框可以获取日期和时间的值，但支持的浏览器不完整。我们测试Chrome和Opera支持，其他浏览器尚未支持。所以，在获取日期和时间，目前还是推荐使用jQuery等前端库来实现日历功能。额外属性和number一致；
6. 当type为color时，实现文本框获取颜色的功能，最新的现代浏览器测试后IE不支持，其余的都能显示一个颜色对话框提供选择；
7. 当type为checkbox、radio时，生成一个获取布尔值的复选框或固定选项的单选框；
8. 当type为submit、reset和button 时，生成一个按钮，三种模式：提交、重置和一般按钮，和<button>元素相同；
9. 当type为image时，生成一个图片按钮，点击图片就实现提交功能，并且传送了分区响应数据。图片按钮也提供了一些额外属性；
10. 当type为 email、tel、url时，email为电子邮件格式、tel为电话格式、url为网址格式。额外属性和text一致。但对于这几种类型，浏览器支持是不同的。email支持比较好，现在浏览器都支持格式验证；tel基本不支持；url支持一般，部分浏览器只要检测到 http://就能通过。
11. 当type为hidden时，生成一个隐藏控件，一般用于表单提交时关联主键 ID 提交，而这个数据作为隐藏存在；
12. 当type为file时，生成一个文件上传控件，用于文件的上传，额外使用accept属性指定接受的MIME类型；
select标签：下拉列表元素至少包含一个<option>子元素，才能形成有效的选项列表。<select>元素包含两个子元素<option>项目元素和<optgroup>分组元素，还包含了一些额外属性。name设定提交时的名称；disabled将下拉列表禁用；form将表单外的下拉列表与某个表单挂钩；size设置下拉列表的高度；multiple设置是否可以多选；autofocus获取焦点；required选择验证，设置后必须选择才能通过；
textarea标签：生成一个可变更大小的多行文本框。额外的属性有rows设置行数，cols设置列数，wrap 设置是否插入换行符，有soft和hard两种。
output标签：就是计算两个文本框之间的值，其实就是内嵌了JavaScript功能。

全局属性和其他

HTML实体就是将有特殊意义的字符通过实体代码显示出来。
meta标签：可以定义文档中的各种元数据，而且一个 HTML 页面可以包含多个<meta>元素。
全局属性：
1.id属性：id 属性给元素分配一个唯一标识符。这种标识符通常用来给CSS和JavaScript调用选择元素。一个页面只能出现一次同一个id名称。
2.class属性：class 属性用来给元素归类。通过是文档中某一个或多个元素同时设置CSS样式。
3.accesskey属性：快捷键操作，windows下alt+指定键，前提是浏览器alt并不冲突。
4.contenteditable属性：让文本处于可编辑状态，设置 true 则可以编辑，false 则不可编辑。或者直接设置属性。
5.dir属性：让文本从左到右（ltr），还是从右到左（rtl）。
6.hidden属性：隐藏元素。
7.lang属性：可以局部设置语言。
8.title属性：对元素的内容进行额外的提示。
9.tabindex属性：表单中按下tab键，实现获取焦点的顺序。如果是-1，则不会被选中。
10.style属性：设置元素行内CSS样式。
